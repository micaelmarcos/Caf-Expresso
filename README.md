CAFÉ EXPRESSO
Status do Projeto: 🚀 Em Desenvolvimento

📝 Descrição
O Café Expresso é uma solução moderna para autoatendimento em estabelecimentos comerciais. Desenvolvido como projeto prático 
para a disciplina de Engenharia de Software Avançada, o sistema funciona em totens, permitindo que o cliente gerencie seu 
próprio pedido de forma intuitiva, desde a consulta de produtos até a finalização do pagamento.

✨ Funcionalidades Principais
Interface de Totem: Design focado em usabilidade e agilidade para o cliente final.

Gestão de Produtos: Cadastro completo e consulta de itens em estoque.

Carrinho Inteligente: Adição de múltiplos itens com cálculo automático de subtotal e total.

Fluxo de Status do Pedido: Rastreabilidade em tempo real:

Pendente ➝ Pago ➝ Em Preparo ➝ Finalizado.

Processamento de Pedidos: Automação do cálculo de valores e fechamento de conta.
